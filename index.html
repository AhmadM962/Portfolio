03 // Research

Academic research paper.

📄 Academic Research Paper · 2025 – 2026
A Lightweight, Privacy-Preserving Offline Phishing Detection System Using Explainable URL-Based Analysis
Computing Research Project — Al Hussein Technical University
Designed and evaluated a fully offline, client-side phishing URL detection system using a lightweight XGBoost classifier trained exclusively on intrinsic URL-based lexical and structural features. Integrated SHAP (SHapley Additive Explanations) for post-hoc interpretability, providing both global feature importance and local per-prediction explanations. Validated through a prototype browser extension performing real-time on-device inference with non-intrusive inline warnings — eliminating all dependence on cloud services or external reputation APIs.
